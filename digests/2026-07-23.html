🌟 极大维度推荐

  1. Mage-Flow: An Efficient Native-Resolution Foundation Model for Image Generation and Editing [2607.19064]
    亮点：4B 规模紧凑生成栈，Mage-VAE 用一步扩散式编解码 + anchor-latent 正则，在保持强公开 VAE 重建质量的同时把 tokenization 成本降低一个数量级以上；配合原生分辨率多模态 DiT（rectified flow matching）统一文生图与指令编辑。
    代表性维度：①突破性 + ③相关度：以极小算力实现原生分辨率生成/编辑，对端侧影像与轻量部署高度相关
  2. Wavefront Parallelization for Efficient Learned Image Compression [2607.19082]
    亮点：无需改结构或重训，借鉴视频编码波前并行思想把自回归上下文模型推理重排为最优交错顺序，在保持精确自回归依赖的前提下把预训练模型（如 Cheng et al.）加速 13× 以上。
    代表性维度：①突破性 + ③相关度：即插即用的推理加速，直接惠及 low-level / codec 端侧落地
  3. ABot-World-0: Infinite Interactive World Rollout on a Single Desktop GPU [2607.19191]
    亮点：动作条件视频世界模型，单张桌面 GPU 实时长时程闭环交互；用双向 teacher 经 teacher forcing + ODE 蒸馏成因果 student，并提出 LongForcing 对齐长自 rollout，配 14 项确定性质检数据流水线。
    代表性维度：①突破性 + ④覆盖面：把交互式世界模型推到消费级硬件实时闭环
  4. OmniReasoner: Thinking with Long Audio-Video via Native Tool Use [2607.19339]
    亮点：长音视频推理的工具调用后训练框架，模型先构建低成本全局预览，再按需对指定时间区间调用 zoom-in 工具做高保真视觉/音频细查，经 SFT+RL 学会『是否/在哪里』放大。
    代表性维度：②影响力 + ④覆盖面：为 omni-modal LLM 长视频推理给出稀疏证据高效检索范式
  5. FilmWorld: Agentic Novel-to-Film Generation through Dynamic Cinematic World Modeling [2607.19038]
    亮点：把小说到电影形式化为动态电影世界建模，分构建（把抽象文学叙事落成有状态持久世界）与生成两阶段，突破现有视频模型只能短单场景片段的局限，支持长时程多场景动态实体状态演化。
    代表性维度：④覆盖面：长篇多场景生成式视频叙事的系统化框架
  6. Appearance Pointers -- Multimodal Region Control of Diffusion Transformers [2607.19344]
    亮点：提出 appearance pointers 紧凑 token，通过区域对应网络把文本/图像输入与用户指定 mask 对齐，指导 DiT 在正确空间位置施加正确外观线索，实现材质/身份/布局的精确区域控制。
    代表性维度：③相关度：面向创作者的可控图像生成/编辑，解决纯文本 prompt 无法精确区域控制的痛点
  7. Text Template Tokens Are Implicit Semantic Registers in Diffusion Transformers [2607.19139]
    亮点：提出因果可解释性框架，结合注意力分解与跨 token/head/layer 的定向干预，发现文生图 DiT 中结构模板 token 虽在编码器输出几乎不含 prompt 信息，却成为主导的图到文注意力汇聚点并因果维持物体身份，充当隐式语义寄存器。
    代表性维度：①突破性：为大规模 DiT 内部机制提供可验证的因果理解
  8. Context-structured Video Anomaly Detection with Large Vision-Language Models [2607.19077]
    亮点：CSI-VAD 免训练视频异常检测，把视频分解为环境/物体/时间三类上下文分支各自推理，仅基于上下文特定视觉线索判断异常，避免整体采样推理漏掉上下文相关线索。
    代表性维度：③相关度：无标注免训练的视频理解，贴合视频处理主题
  9. ERank in Latent Space as an Image-Complexity and Richness Measure [2607.19315]
    亮点：用冻结预训练编码器单次前向得到特征图通道协方差的有效秩（ERank），作为逐样本、无标签的视觉丰富度度量；与码率/锐度/边缘密度相关，与人类复杂度标注相关 r=0.72，可作数据筛选（去低 ERank 提升超分、去高 ERank 提升 OCR）。
    代表性维度：③相关度：轻量可解释的图像复杂度度量，对 low-level 数据筛选实用
  10. Continual Video-MLLM Adaptation over Evolving Domains [2607.18716]
    亮点：提出 Distribution-Aware Expert Routing（DAER），为持续演化域上的视频多模态大模型做参数高效持续适配，用域隔离专家避免共享适配空间导致的跨域干扰与灾难性遗忘。
    代表性维度：②影响力：面向真实部署中异构视频流的持续学习方案
