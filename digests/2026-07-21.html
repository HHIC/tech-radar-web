🌟 极大维度推荐

  1. FlowMimic: Mask-free Visual Editing and Generation with Pixel-pair Warped Flow Field for Online Video Editing Data Generation and Modality Mimicry [2607.18227]
    亮点：视频编辑数据获取长期依赖繁重的对象掩码标注+I2V/ControlNet 合成配对+VLM 过滤，任务可扩展性差；FlowMimic 用像素对扭曲的光流场实现无掩码的图像/视频统一编辑与生成，并在线生成视频编辑训练数据，显著拓宽编辑任务多样性。
    代表性维度：①突破性：绕开掩码标注瓶颈，把图像编辑的任务丰富度迁移到视频编辑
  2. Consistent Feature Transport for Image Relighting [2607.17833]
    亮点：扩散式重打光在复杂光照下常出现光照不稳或内容不一致，因缺乏对图像间特征变换的显式建模；作者把重打光重构为'光照特征传输'问题，基于 rectified flow 提出 CFT 训练原则显式约束源→目标分布的光照一致传输。
    代表性维度：①突破性：以特征传输视角统一重打光，改善光照变化稳定性与内容保真
  3. HOMIE: Human-object Centric Video Personalization via Multimodal Intelligent Enhancement [2607.18217]
    亮点：人-物中心的视频个性化难在既保主体保真又保人物交互模式（尤其 logo 等抽象物体），且难利用 OCR/多视角等主体内参考；HOMIE 用多模态智能增强兼顾主体保真与交互准确并挖掘主体内潜在对应。
    代表性维度：③相关度：主体驱动视频生成，贴合视频编辑/生成方向
  4. When 2D Cues Fail: Improving Image Manipulation Localization with Reliable 3D Geometry [2607.18040]
    亮点：篡改区域与周边外观高度融合时，噪声痕迹/低层伪影等 2D 取证线索失效；作者引入单目重建得到的 3D 几何一致性，指出篡改区域即便局部外观自洽也会违反场景几何结构，从而提升图像篡改定位鲁棒性。
    代表性维度：③相关度：图像取证/低层视觉，几何推理提供新判据
  5. Empowering On-Device Model Adaptation with an Edge AI Inference Accelerator [2607.18101]
    亮点：端侧终身个性化受限于算力/功耗/内存，端到端反传不现实；作者复用商用推理加速器 Hailo-8L 做冻结骨干的 INT8 特征提取、仅在 host CPU 微调轻量 FP32 分类头，构成异构端侧适配流水线。
    代表性维度：③相关度：端侧/手机部署视角，给出可落地的低成本在设备学习方案
  6. ConsiSpace: Learning Geometric Consistency Matters for Video Spatial Reasoning [2607.17599]
    亮点：视频空间推理需在长时程、视角变化下聚合一致空间证据，但现有 MLLM 偏语义、难从冗余观测里稳定聚合几何证据；ConsiSpace 把空间一致性同时用作证据组织原则与几何敏感推理框架。
    代表性维度：②影响力：切中 MLLM 空间推理弱项，几何一致性思路可迁移
  7. OrientSAM: Mitigating Camera-Centric Shortcut in Multimodal Spatial Reasoning via Orientation-Aware Spatial Alignment [2607.17657]
    亮点：MLLM 空间推理常依赖相机中心线索而非从参考物体视角推理，导致非相机参考系下系统性错误；作者定位到物体朝向是该捷径的关键，提出注入显式朝向信息的朝向感知空间对齐 OrientSAM。
    代表性维度：①突破性：诊断并纠正相机中心捷径，提升透视变换类空间推理
  8. ST-Veto: Spatio-Temporal Token Veto for Diffusion MLLMs via Taylor Prediction and Visual Grounding [2607.17884]
    亮点：扩散式多模态 LLM 可无序解码、支持迭代自修正，但推理机制欠探索；ST-Veto 训练无关，利用每步可观测全部 token 位置，不只看当前步置信度，而是否决时序不稳定的 token 以增强推理。
    代表性维度：②影响力：为扩散 MLLM 的推理增强与效率给出免训练手段
  9. TimeLens2: Generalist Video Temporal Grounding with Multimodal LLMs [2607.17423]
    亮点：视频 MLLM 能描述发生了什么却难定位证据何时出现；TimeLens2 面向通用视频时序定位，将时序证据全程建模为区间集合，解决变基数、跨时长/域/视角的证据区间预测与 RL 奖励错配问题。
    代表性维度：③相关度：视频理解/处理，时序证据集合建模有通用性
  10. Exploration Matters for Escaping the Blur Trap in 3D Gaussian Splatting [2607.17965]
    亮点：3DGS 显式建模引入优化梯度偏置，非凸优化易陷入被作者称为'模糊陷阱'的局部次优；通过严格数学分析并把简单显式探索融入 3DGS 优化，帮助逃离模糊陷阱以提升重建质量。
    代表性维度：①突破性：定位并缓解 3DGS 的一个基础性优化缺陷
  11. Multi-Dimensional Quality Assessment for AI-Generated Human-Centric Videos: Dataset and Model (HVEval+) [2607.16742]
    亮点：AIGC 人物视频常有质量/语义不匹配问题；HVEval+ 在 HVEval 基础上加成对偏好标注，含 1k prompts、24 个 T2V 模型生成的 20k 视频与 60k MOS+60k 偏好，成为最大的整体质量评估数据集。
    代表性维度：④覆盖面：为 AIGC 视频质量评估提供大规模基准与模型
