🌟 极大维度推荐

  1. MonkeyOCRv2: A Visual-Text Foundation Model for Document AI [2607.11562]
    亮点：构建迄今最大文档图像预训练语料 MonkeyDoc v2(1.13 亿图、17 种语言),用'图到文生成 + 像素级文档重建'联合预训练,兼顾语义对齐与字符笔画级细粒度感知。
    代表性维度：②影响力/④覆盖面:文档 AI 基座模型,语料规模与任务定位具行业代表性。
  2. X-Lens: Real-Time Metric Depth Estimation with Heterogeneous Cameras [2607.12993]
    亮点：面向鱼眼+针孔混合多视图的前馈度量深度模型,用可学习标定 token 做粗对齐、Jacobian 参数化畸变偏置注入 cross-attention,仅 0.04B 参数即可 41FPS 输出稠密深度与全局尺度。
    代表性维度：①突破性/③相关度:端侧实时、异构相机几何感知,与手机多摄影像高度相关。
  3. Color Pass-Through via Camera-Display Coupling [2607.12746]
    亮点：针对手机'拍摄→屏显'色彩/亮度/对比失真,提出端到端学习框架直接在采集图上建模,打破相机与显示分段标定、低维色变连接造成的信息瓶颈与误差累积。
    代表性维度：③相关度:直击手机影像/ISP-显示链路的实际痛点,工程价值突出。
  4. RFMSR: Residual Flow Matching for Image Super-Resolution [2607.12753]
    亮点：超分中不再从纯高斯先验出发,而用残差流匹配保留低质输入已有结构信息,避免 T2I 大模型的庞大体量与训练开销,兼顾感知质量与效率。
    代表性维度：③相关度/①突破性:low-level CV / 图像增强的高效生成式路线。
  5. VisCo: Leveraging Large Language Models as Intrinsic Encoders for Visual Token Compression [2607.12756]
    亮点：指出高压缩比下有效视觉 token 压缩取决于强信息编码——这一能力预训练 VLM 本已具备但被忽视,遂利用 VLM 自身作内在编码器压缩,避免外接模块的重训练成本与先验损伤。
    代表性维度：②影响力:VLM 推理时延/显存瓶颈的通用优化方向。
  6. UniVR: Thinking in Visual Space for Unified Visual Reasoning [2607.12800]
    亮点：首次尝试从纯视觉演示中同时学复杂推理、细粒度物理动态与长程规划;提出 VR-GRPO(全局+步级奖励)的强化学习范式,无需图文对与任务启发式,并构建 VR-X 基准。
    代表性维度：①突破性:纯视觉空间统一推理的新范式探索。
  7. DynTrace: Tracking Dynamic Object Evidence for 4D Spatio-Temporal Reasoning in MLLMs [2607.12503]
    亮点：面向 MLLM 的 4D 时空推理,针对稀疏帧观测割裂动态线索、难区分真实物体运动与相机诱发表观运动的问题,引入动态物体证据跟踪以支撑连贯 4D 推理。
    代表性维度：③相关度:多模态大模型在动态场景/具身交互的能力短板补强。
  8. Beyond the Eye: Efficient Multimodal Reasoning via Self-Regulated Implicit Visual Tools [2607.11106]
    亮点：针对'Thinking with Images'频繁外部工具调用与图像重编码的高开销,提出 BEE 隐式视觉工具范式,将工具调用行为直接纳入训练目标,培养模型自调节的调用机制。
    代表性维度：②影响力:多模态细粒度推理的效率-延迟优化。
  9. Actor as Its Own Critic: Unifying Region Understanding and Localization via CycleGRPO [2607.11581]
    亮点：利用区域理解与定位的对偶性构建'region→text→region'自评估强化学习(CycleGRPO),单个 MLLM 既作 actor 生成区域描述又作 critic 将文本重新落回空间,仅需区域输入。
    代表性维度：①突破性:统一理解-定位的自洽 RL 框架。
  10. Do We Really Need Multimodal Emotion Language Models Larger Than 1B Parameters? [2607.12787]
    亮点：质疑多模态情感识别必须 7B+ 的惯例,系统探讨 1B 以下模型能否保质,面向机器人/手机等资源受限平台的实时部署。
    代表性维度：③相关度:端侧高效多模态,呼应手机/边缘部署需求。
  11. Hy-Embodied-VLM-1.0: Efficient Physical-World Agents [2607.12894]
    亮点：面向具身智能体的高效基座模型,定义'动作相关状态理解→动作转移推理→序列与自适应推理'的动作中心能力分级,从预训练阶段培育具身能力。
    代表性维度：④覆盖面:LLM/具身 Agent 主题的代表性工作。
