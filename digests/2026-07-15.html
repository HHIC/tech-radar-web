🌟 极大维度推荐

  1. IQA-T1: Tool-based Visual Evidence Reasoning for Image Quality Assessment [2607.12375]
    亮点：针对 MLLM 做图像质量评估时依赖语义先验、对低层退化不敏感的问题,提出工具化视觉证据推理框架,让模型显式调用感知观测工具取证再判分,提升开放世界 IQA 的泛化与可解释性。
    代表性维度：③相关度——直接落在 low-level CV + 图像质量,与手机影像/图片处理主题簇高度契合
  2. RFMSR: Residual Flow Matching for Image Super-Resolution [2607.12753]
    亮点：指出主流 T2I 扩散超分模型体量巨大、训练昂贵,提出残差流匹配:不再从纯高斯先验传输到数据分布,而是围绕低分辨率残差建流,以纯视觉小模型实现高感知质量超分。
    代表性维度：③相关度——图像复原/超分的实用化路线,贴合图片处理与 low-level CV
  3. Color Pass-Through via Camera-Display Coupling [2607.12746]
    亮点：针对手机『拍到的』与『屏上看到的』颜色/亮度/对比明显不一致,提出把相机-显示端到端耦合建模(而非分别标定再低维色彩变换),打破两段式信息瓶颈实现色彩直通。
    代表性维度：③相关度——正中手机拍照/ISP/手机影像主题簇,少见的端到端 capture-to-display 研究
  4. MambaPSA: A Mamba-based Replacement for C2PSA in YOLO26 [2607.12681]
    亮点：把状态空间模型 Mamba 引入最新免-NMS 检测框架 YOLO26,以线性复杂度的 MambaPSA 替换主干末端 C2PSA 块,并在 neck 多尺度插入双向 Vision Mamba,做轻量高效检测。
    代表性维度：①突破性——SSM 替代自注意力在主流检测器上的落地尝试,工程价值高
  5. AVQ-Attention: Adaptive Vector-Quantized Attention [2607.12789]
    亮点：针对 VQ 注意力用固定码本容量、导致高注意力区被粗糙近似而低注意力区浪费容量的问题,提出按注意力质量分布自适应分配码本容量,把 O(N²) 降到 O(MN) 且更省得其所。
    代表性维度：①突破性——高效注意力的一般性改进,可迁移到多种 transformer
  6. VisCo: Leveraging Large Language Models as Intrinsic Encoders for Visual Token Compression [2607.12756]
    亮点：VLM 视觉 token 过多带来推理延迟与显存开销;VisCo 不加外部压缩模块、也不靠启发式度量,而是复用 LLM 自身作为内在编码器完成视觉 token 压缩,兼顾高压缩比与性能。
    代表性维度：②影响力——多模态推理效率是当前热点,方法通用易被采用
  7. DynTrace: Tracking Dynamic Object Evidence for 4D Spatio-Temporal Reasoning in MLLMs [2607.12503]
    亮点：指出 MLLM 在连续动态场景、尤其跟踪动态物体证据上仍薄弱,提出以动态物体证据追踪支撑连贯的 4D(3D 空间+时间演化)时空推理,面向具身交互。
    代表性维度：④覆盖面——多模态大模型向动态/具身场景延伸的代表方向
  8. Hy-Embodied-VLM-1.0: Efficient Physical-World Agents [2607.12894]
    亮点：面向物理世界具身智能体的高效基座模型,从预训练阶段起就围绕『以动作为中心』的能力分类体系培养感知、推理与环境交互能力。
    代表性维度：②影响力——具身 VLM + Agent 交叉,连接多模态与自主智能体两大主题
  9. Beyond the Eye: Efficient Multimodal Reasoning via Self-Regulated Implicit Visual Tools [2607.11106]
    亮点：『Thinking with Images』范式频繁外部工具调用与重复图像重编码带来高开销;BEE 提出自调节的隐式视觉工具,减少显式工具调用与重编码,降低延迟。
    代表性维度：①突破性——对当红『看图思考』范式的效率再造
  10. The Seriality Gap in Video Diffusion Models [2607.13031]
    亮点：用多球硬球碰撞的受控实验证明:标准双向视频扩散在因果链变长时性能显著退化(加去噪步数也无救),而单球对照组退化消失,精确分离出『依赖事件结构』而非视频长度才是瓶颈。
    代表性维度：①突破性——揭示视频生成模型物理因果推理的系统性缺陷,诊断性强
  11. X-Lens: Real-Time Metric Depth Estimation with Heterogeneous Cameras [2607.12993]
    亮点：面向鱼眼+针孔混合、数量可变的标定相机,用可学习标定 token 做粗对齐、雅可比参数化畸变偏置注入交叉注意力,前馈实时估计度量深度。
    代表性维度：④覆盖面——多相机几何感知,贴合手机多摄影像系统
